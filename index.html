Our Work

Questionable.io

AZPixels created Questionable.io to fill a void among Web Applications. Questionable.io is the only multiple-choice test tool that allows for writing code in the questions and answers with the quality of a real code editor.

CTEAZ

This was a long term project starting in 2009 and ending in 2018 to create a test management platform for the Arizona Department of Education (Career and Technical Education Department).

CSSDay.io (2015)

CSSDay.io is (was) a developer conference for programmers to be better acquainted with CSS. AZPixels was proud to plan the event including doing the website.

CSSDay.io (2016)

It's back again! We did CSSDay two years in a row and it was a blast again to make the website an participate in community events.

Redstone Capital Management
